For firewall fuel filter maintenance and inspections see Cross Reference Placeholder.

The engine is equipped with a disposable fuel filter on the high-pressure pump located at the 3 o'clock position on the accessory case. This element is an AN6235-3A. The filter should be changed anytime the firewall filter is found contaminated or every 300 hours. See the proper engine manual in Cross Reference Placeholder details on removing and replacing this filter.

CAUTION

If any contamination is found in either fuel filter, investigate the cause and rectify the problem.
