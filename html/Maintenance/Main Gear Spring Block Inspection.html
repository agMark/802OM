In the event of a hard landing or an obstacle collision, the main landing gear spring blocks should be checked for straightness and damage. The original (as manufactured) dimensions of the block are shown in

Inspection:

  1. Check for any visible signs of damage, particularly around bolts / holes and interface with the gear spring.
  2. Using a straight edge, check the exterior of the block for straightness. The entire block should be straight within .040 in along its entire length.
