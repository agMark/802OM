The ferry fuel system utilizes the hopper as a fuel tank (see Fig Cross Reference) and is not an FAA Certificated fuel system. However, it may be installed provided a logbook entry is made for both the installation and removal.

Great care must be taken to ensure that the ferry fuel system is installed and removed properly and that contaminants are not introduced into the fuel system.

The ferry fuel system should be installed per Drawing 50280, Sheet 1 in firefighting aircraft and 50280, Sheet 3 for agricultural aircraft. The installation and removal of the ferry fuel system may be done in accordance with these drawings or in accordance with the instructions provided in this section which are provided here for convenience.

Installation of Ferry Fuel System in Agricultural Aircraft (Ref Dwg 50280, Sheet 3)

  1. Ensure that hopper is completely empty, clean and dry before beginning installation.
  2. Remove existing plug from fuel valve and retain for reinstallation.
  3. Drill 1.5 inch hole in belly skin at the location shown below.
  4. Disconnect hopper rinse line at the point that it enters the hopper or adapter. Install 51260-1 hose on line that does not have hose. Install 51887-1 plugs into end of each hose (2). Secure hose and plugs with WS200M16S clamps. Secure lines as required with cable ties.
  5. Build up plug / adapter and pickup hose assembly.
    1. Attach 51885-1 screen to 51886-1 pickup hose with a Q200M16S clamp.
    2. Install flared fitting end of pickup hose onto internal end of nipple in the 51884-1 plug assembly.
    3. Install 80376-2 adapter hose over plug assembly and secure with two Q200M56S clamps.
  6. Install hopper / hose adapter and pickup hose & screen inside of the gatebox.
    1. Secure hose and screen with cable ties at desired height.
    2. Secure 80376-2 adapter hose onto gatebox tube with two Q200M56S clamps.
  7. Connect supply hose from the hopper / hose adapter to the fuel valve routing through the 1.5 inch hole in the belly skin with a MS35489-135 grommet. Secure lines where required and able with cable ties.
  8. Mark fuel valve handle position for ferry fuel. Placard position depends on fuel valve type. For 802-102 and subsequent with AVI valve, install placard on cockpit side skin. Contact Air Tractor customer support otherwise.
  9. If desired, or required, install hopper lid fuel cap by cutting a 3.5 inch hole in the hopper lid and installing filler cap with 12 AN4C6A bolts, AN960C4 washers and MS21044C4 nuts. Install 416-50 retention chain betwwen filler cap and cap installation bolt as shown. Seal all around inside of cap with PR-1422 sealant.
  10. Coat all of the 80158 vent hoses and attaching clamps that are located inside the hopper with PR-1422 sealant to prevent fuel from getting to the hoses.
  11. Seal hopper lid with PR-1422 sealant.
  12. Allow all sealant to cure for at least 12 hours before flight.
  13. Make a logbook entry stating that the ferry fuel system has been installed as a "TEMPORARY INSTALLATION FOR FERRY FLIGHT ONLY. MUST BE REMOVED BEFORE RESTRICTED OPERATION RESUMED."

Removal of Ferry Fuel System

  1. Turn fuel valve pointer to off position.
  2. Remove hose and fitting at fuel valve and install plug.
  3. Replace inspection cover in belly skin. Remove coupler, fittings, screen and filter from gatebox.
  4. Remove plug from bottom load tube opening in gatebox and install tube.
  5. Install spray pump, spray plumbing, and brake cable in reverse order of removal.
