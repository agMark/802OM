The Trotter Controls Gen II FRDS was installed at the factory between 2007 and 2020.  This system was comprised of a programable computer and a hydraulically powered gatebox system.  A description of the Gen II FRDS is provided in Trotter Controls Manuals 9001-0011 and 9001-0012.

The Trotter Controls Gen III FRDS replaced the Gen II beginning in 2020.  This system is similar to the Gen II system except that hydraulics are replaced by a fully electric gatebox control system.  A description of the Gen III FRDS is provided in Trotter Controls Manuals 9001-0056, 9001-0057.
